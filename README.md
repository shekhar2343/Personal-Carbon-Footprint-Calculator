> A smart web app that calculates your carbon footprint and helps you reduce it with AI-powered suggestions 🌍

---

## 🚀 Features

- 🌍 Calculate personal carbon footprint  
- 📊 Interactive charts using Chart.js  
- 🤖 AI-based eco-friendly suggestions  
- 💾 Local storage to save results  
- 📈 Compare with global averages  
- 💬 Chatbot integration  

---

| CSS | Express.js | GROQ API |

---

## ⚙️ Installation

```bash
git clone https://github.com/shekhar2343/Personal-Carbon-Footprint-Calculator.git
cd Personal-Carbon-Footprint-Calculator
npm install
```

---

## ▶️ Run Locally

```bash
npm start
```

Open in browser:
```
http://localhost:3000
```
